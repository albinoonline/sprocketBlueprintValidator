PATCH NOTES:

1.0.0 (2/3/23)

Tool Update:

This is no longer considered a pre-release, though there are still some things to improve (mainly plate occlusion, recursive turrets).

Updated Easter Egg.

Compartments named "deco" are no longer counted for armor/size.

You can now see all plates below certain user set thicknesses, and more views are generated.

1/17/23

Tool Update:

Armor Order is now front side rear, rather than front rear side.

Tool now reads twin transmission option.

Stat breakdowns are now downloadable as a CSV file, and multiple tanks can now be read at once

12/30/22

Tool Update:

A canvas (generated image) of the tank is now created, this image shows the weakest plate frontally, or a plate that is tied for the weakest, note this does not display the turret ring, which may be weaker than the toughest plate shown

The turrets second side armor is now rounded to the nearest hundredth (whoops..., and yes I haven't bothered learning which side is left or right in the code).

12/28/22

T.E.A.B.A.G. 10 Rule Clarification/Tool Update:

On the issue of the bottom turret plate, and weaker roof armor under the turret; Hull roof armor (or turret roof if you want a pancake turret) must be to spec (15mm), but the underside of the turret no longer has to be, the tool will now ignore plates that are both on a turret, and whoos points "up" value are exactly 0 (you can check by selecting the point, and opening the position tab).

Tool Update:

Output values are now rounded to the nearest hundredth.

Width now says raw is hull and simplified is track.
